## CROSS-PLATFORM RULES

1. **NEVER copy-paste the same copy across platforms** — always adapt for each (-2 Clarity penalty)
2. **Every post on every platform must include https://www.puretask.co** (-2 Clarity penalty if missing)
3. **Every post must have an image or video** (-3 Conversion penalty if missing)
4. **City posts must have city-specific visuals** (-2 Relatability penalty if generic)
5. **Instagram: trim hashtags to 5 in Ayrshare API call** (plan limitation — keep full list in DB)
6. **TikTok requires video** — never post text-only to TikTok
7. **Space posts minimum 3 hours apart per platform** to avoid feed flooding
